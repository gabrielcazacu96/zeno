Zeno is built on a carefully selected stack of modern, production-ready technologies.

## Core

- **[TypeScript 5](https://www.typescriptlang.org)** — JavaScript with syntax for types
- **[React 19](https://react.dev)** — The library for web and native user interfaces
- **[Next.js 16](https://nextjs.org)** — The React Framework for the web

## Build & Dev Tools

- **[Turborepo](https://turbo.build/repo)** — Monorepo management & high-performance build system
- **[Turbopack](https://turbo.build/pack)** — High-performance TypeScript bundler
- **[Ultracite](https://ultracite.ai)** — JavaScript/TypeScript linting and formatting

## Styling

- **[Tailwind CSS 4](https://tailwindcss.com)** — Utility-first CSS framework
- **[Shadcn UI](https://ui.shadcn.com)** — Beautifully designed components

## Database & Backend

- **[Supabase](https://supabase.com)** — Open source Postgres database, Auth, APIs, Edge Functions, Realtime, Storage
- **[Drizzle](https://orm.drizzle.team)** — TypeScript ORM for SQL databases

## Testing

- **[Vitest](https://vitest.dev)** — Next generation testing framework
- **[Playwright](https://playwright.dev)** — End-to-end testing for modern web apps
