The Zeno CLI provides commands for creating, managing, and deploying Zeno applications.

## Installation

```bash
pnpm add -g @zeno/cli
```

## Commands

### Create a new project

```bash
zeno create my-app
```

### Add a component

```bash
zeno add button
```

### Generate code

```bash
zeno generate model User
```

See the [Installation](/docs/cli/installation) guide to get started with the Zeno CLI.
