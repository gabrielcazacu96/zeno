## Prerequisites

- Node.js 18 or later
- pnpm 8 or later

## Installation

Install the Zeno CLI globally:

```bash
pnpm add -g @zeno/cli
```

Or use it directly with npx:

```bash
npx @zeno/cli create my-app
```

## Verify Installation

```bash
zeno --version
```
