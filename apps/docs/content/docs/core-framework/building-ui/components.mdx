Zeno ships a set of unstyled-but-themed React primitives in the
`@zeno/ui` package. They mirror [shadcn/ui](https://ui.shadcn.com)
1:1 in API and naming and are built on top of
[Base UI](https://base-ui.com).

```tsx
import { Button } from "@zeno/ui/button"
import { Dialog, DialogContent, DialogTrigger } from "@zeno/ui/dialog"
```

## Anatomy

- Composition via Base UI's `render` prop — pass another element to swap the underlying tag (e.g. render a `Button` as a Next.js `Link`)

```tsx
import Link from "next/link"
import { Button } from "@zeno/ui/button"

<Button render={<Link href="/recover">Recover</Link>} variant="link" />
```

## Theming

Theme tokens live in `@zeno/ui/styles/theme.css`. Import it once in
your consuming app's global stylesheet:

```css
@import "@zeno/ui/styles/theme.css";
```

See [Theming](/docs/core-framework/building-ui/theming) for token
overrides and dark mode.
