## Requirements

### Package Manager

Zeno uses [pnpm](https://pnpm.io) for fast, disk-efficient package management. Install it globally:

```bash
npm install -g pnpm
```

### Node.js

Zeno requires Node.js 24 or higher. Install it from the [Node.js website](https://nodejs.org/en/download/).


## Quick Start

Clone the Zeno repository:

```bash
git clone https://github.com/your-org/zeno.git
cd zeno
```

Install dependencies:

```bash
pnpm install
```

Start development servers:

```bash
pnpm dev
```

## Verify Installation

Run the full CI pipeline to ensure everything works:

```bash
pnpm ci
```

This runs:
1. Linting with Ultracite
2. TypeScript type checking
3. Build all packages
4. Unit tests with Vitest
5. E2E tests with Playwright
