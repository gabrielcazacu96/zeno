## Philosophy

All that Zeno does follows these principles:

**Development Speed:** We aim to accelerate web application development through intelligent 
defaults, streamlined workflows and high-level abstractions.

**Development Experience:** We believe that development experience is part of development productivity.
Spending time on development experience is less time spent on building features. Let us do it for you.

**Opinionated:** In order to achieve the goal of development speed & experience, we take strong stances on 
development best practices and architectural decisions. Rather than presenting endless configuration 
options, we carefully choose proven patterns and technologies that work well together. This 
opinionated approach eliminates decision fatigue, reduces cognitive overhead, and helps teams maintain 
consistency across projects. While we provide flexibility where it matters, we believe that having a 
clear, well-defined path leads to better, more maintainable applications.

**High Level:** Being opinionated allows us to provide a high level of abstraction, meaning you can 
focus on building features rather than worrying about re-implementing the wheel & the underlying
implementation details.

**AI Conscious:** AI is revolutionizing how we build web applications. Zeno embraces this by putting it at 
the core of the development experience. Through AI-first principles, we are creating a framework that 
seamlessly integrates with AI to help you build better applications even faster.

**Open Code:** Zeno embraces the open code distribution strategy popularized by Shadcn. When relevent,
we provide source code directly into your codebase. This empowers developers and AI with more control 
over the codebase.

Let's dive into how Zeno achieves these goals.

### Modern

Zeno embraces the latest and greatest technologies to ensure your applications are built on a solid, 
future-proof foundation.

- **[TypeScript 5](https://www.typescriptlang.org)**: JavaScript with syntax for types.
- **[Turborepo](https://turbo.build/repo)**: Monorepo management & high-performance build system for TypeScript codebases.
- **[Turbopack](https://turbo.build/repo)**: High-performance TypeScript bundler.
- **[React 19](https://react.dev)**: The library for web and native user interfaces.
- **[Next.js 16](https://nextjs.org)**: The React Framework for the web.
- **[Tailwind CSS 4](https://tailwindcss.com)**: Utility-first CSS framework for rapid UI development.
- **[Ultracite](https://ultracite.ai)**: JavaScript/TypeScript linting and formatting.
- **[Supabase](https://supabase.com)**: Open source Postgres database, Authentication, instant APIs, Edge Functions, Realtime subscriptions, Storage, and Vector embeddings.
- **[Playwright](https://playwright.dev)**: Next-generation end-to-end testing for modern web apps.


### Complete

Zeno is a complete solution for building web applications. Here is a non-exhaustive list of what it includes:

- **Monorepo**: Scalable codebase management with shared configurations and packages
- **Database**: Full-featured Postgres database with real-time capabilities
- **API**: Built-in API routes with Next.js App Router
- **Authentication**: Secure user authentication with multiple providers
- **Authorization**: Role-based access control and permissions
- **Storage**: File storage with CDN integration
- **Forms**: Form handling with validation and error management
- **UI**: Beautiful, accessible, and customizable components
- **Layouts**: Pre-built layouts for common application patterns
- **Testing**: Comprehensive testing suite with Vitest and Playwright
- **CI/CD**: Automated deployment pipelines with GitHub Actions
- **AI**: AI-powered development tools and integrations
- **DX**: Enhanced development experience with pre-configured tools
- **I18N**: Internationalization and localization support
