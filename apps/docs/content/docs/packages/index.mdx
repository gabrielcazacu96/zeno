Zeno provides several shared packages that can be used across your applications.

## Available Packages

### @zeno/ui

React component library built on Shadcn UI with beautiful, accessible components.

[View documentation](/docs/packages/ui)

### @zeno/supabase

Supabase integration utilities for Next.js applications.

[View documentation](/docs/packages/supabase)

### @zeno/tailwind

Shared Tailwind CSS configuration for consistent styling.

[View documentation](/docs/packages/tailwind)

### @zeno/typescript

Shared TypeScript configurations for different project types.

[View documentation](/docs/packages/typescript)
