The `@zeno/tailwind` package provides a shared Tailwind CSS configuration for consistent styling across all Zeno applications.

## Usage

Import the shared styles in your application's CSS:

```css
@import "@zeno/tailwind/globals.css";
```

## Features

- Pre-configured color palette
- Custom utility classes
- Animation utilities
- Typography presets
