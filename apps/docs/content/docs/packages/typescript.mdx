The `@zeno/typescript` package provides shared TypeScript configurations for different project types.

## Available Configurations

### Base Configuration

```json
{
  "extends": "@zeno/typescript/base.json"
}
```

### Next.js Configuration

```json
{
  "extends": "@zeno/typescript/nextjs.json"
}
```

### React Configuration

```json
{
  "extends": "@zeno/typescript/react.json"
}
```

### Strict Configuration

```json
{
  "extends": "@zeno/typescript/strict.json"
}
```

### Strictest Configuration

```json
{
  "extends": "@zeno/typescript/strictest.json"
}
```
