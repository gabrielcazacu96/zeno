The `@zeno/ui` package provides a collection of beautiful, accessible React components built on top of Shadcn UI.

## Installation

The package is already included in your Zeno project. Import components directly:

```tsx
import { Button } from "@zeno/ui/button"
import { Card, CardContent, CardHeader, CardTitle } from "@zeno/ui/card"
```

## Available Components

The package includes all Shadcn UI components plus additional Zeno-specific components.

See the [Components](/docs/framework/components) section in the Framework documentation for usage examples.
