## Philosophy

**Speed oriented:** Zeno was created with a singular focus: to dramatically accelerate web application 
development. This commitment to speed isn't just about rapid development - it's about intelligent 
defaults and streamlined workflows. We've carefully curated the best tools and practices, making 
strategic choices that let you focus on building rather than configuring. Every feature, every 
integration, and every abstraction is designed to help you ship production-ready applications in 
record time.

**Opinionated:** Zeno takes a strong stance on development best practices and architectural decisions. 
Rather than presenting endless configuration options, we've carefully chosen proven patterns and 
technologies that work well together. This opinionated approach eliminates decision fatigue, reduces 
cognitive overhead, and helps teams maintain consistency across projects. While we provide flexibility 
where it matters, we believe that having a clear, well-defined path leads to better, more maintainable 
applications.

**High Level:** Zeno aims to be a high level framework that provides everything you need to build modern web 
applications out of the box. From database management and authentication to UI components and testing 
infrastructure, Zeno eliminates the need to piece together disparate tools and libraries. This 
comprehensive approach means you can start building features immediately without spending time on setup 
and integration.

**AI-Ready:** AI is revolutionizing the development landscape, fundamentally transforming how we build 
web applications. Zeno embraces this transformation by putting AI at the core of the development 
experience. Through AI-first principles and Vibe Coding methodology, we've created a framework that 
seamlessly integrates with artificial intelligence to enhance your development workflow, helping you 
build better applications faster and more efficiently.

**Open Code:** Zeno embraces, when relevent, the open code distribution strategy popularized by Shadcn, 
taking a fundamentally different approach from traditional npm packages. Instead of providing pre-built 
modules, we give you direct access to the source code itself. This philosophy empowers developers with 
more control over their codebase, enabling deep customization and optimization. Moreover, this approach 
aligns perfectly with modern AI-driven development, allowing artificial intelligence to analyze, 
understand, and modify your codebase seamlessly when needed.
